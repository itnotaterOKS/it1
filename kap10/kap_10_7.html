Hva er en array?

En array er det vi kaller en datakolleksjon: En samling av variabler. Så langt har vi sortert all data i enkeltvariabler, noe som er greit når vi ikke har mye data og når vi ikke skal gjenta funksjoner for flere variabler, men som blir tungvint hvis vi for eksempel skal finne gjennomsnittet av 100 tall.

I php opererer vi med to typer arrays: indekserte arrays (minner om arrays i JavaScript) og assosiative arrays (minner om objects i JavaScript).

Vi oppretter arrays i php på samme måte som vi oppretter enkeltvariabler: $arrayNavn = ;
Verdien legger vi til i klammeparenteser og skiller med komma: $arrayNavn = [verdi1, verdi2, verdi3];
Alternativt kan bokas notasjon brukes: $arrrayNavn = array(verdi1, verdi2, verdi3);

Indekserte arrays

I en indeksert array, referer vi alltid til den plassen (indexen) en verdi har i arrayen. Hvis vi har en array med fire verdier, har den første verdien index 0 og den siste verdien index 3. Denne arrayen har lengde 4.

Oppgave

Lag to arrayer: en array navn som inneholder navn på familiemedlemmene dine og en array alder som inneholder alderen deres. Skriv ut count($navn) og count($alder) (ved bruk av echo) samt var_dump($navn) og var_dump($alder) (trenger ikke bruke echo). Hva får du ut?

Metodene count og var_dump, å skrive ut enkeltvariabler fra en indeksert array

count($arrayNavn) gir ut antall elementer i arrayen. var_dump($arrayNavn) gir ut all informasjon om arrayen. I tillegg kan vi skrive ut enkeltverdier i arrayen med
echo $navn[0]."< br >";
echo $navn[count($navn)-1]."< br >";

Å løkke gjennom en array

Når vi har større datasett er det tungtvint å skrive ut én og én verdi fra arrayen (da kunne vi jo like gjerne laget enkeltvariabler). I stedet kan vi løkke gjennom arrayen og skrive ut $arrayNavn[i]:
I løkken
   for ($i = plassen første element; $i <= plassen til siste element; $i++){
     // kode
   }
Vil plassen til første element være 0 og plassen til siste element være lengde-1. Løkken blir dermed
   for ($i = 0; $i < count($arrayen); $i++){ // eller for ($i = 0; $i <= count($arrayen)-1; $i++){
     // kode
   }

Eksempel: skrive ut én og én verdi
   $a = [1, 4, 11, -3, 5, 7];
   var_dump($a);
   for($i = 0; $i < count($a); $i++){
     echo $a[$i]."< br >";
   }

Eksempel: finne det største tallet i arrayen
   $storst = $a[0]; // antar at den største verdien står på plass 0
   for($i = 0; $i < count($a); $i++){
     if($a[$i] > $storst){ // sjekker om det finnes en større verdi
       $storst = $a[$i]; // og erstatter i så fall storst med denne
     }
   }
   echo "Det største tallet i arrayen er $storst < br >";

Assosiative arrays

Mens vi i indekserte arrays refererer til indexen til en verdi, gir vi i assosiative arrays hver verdi en tekst som vi senere refererer til for å hente ut igjen variablen. Vi oppretter assosiative arrays på samme måte som indekserte arrays, men i stedet for å legge inn kun verdiene må vi også legge inn en nøkkel ved bruk av "nøkkel"=>"verdi":
   $familie = ["Asta"=>"mor", "Christian"=>"far", "Iver"=>"barn"];
   var_dump($familie);
   echo "Asta er ".$familie['Asta']."< br >";

Skal vi kunne hente ut navnene må nøkkel og verdi være omvendt:
   $familie = ["mor"=>"Asta", "far"=>"Christian", "barn"=>"Iver"];
   var_dump($familie);
   echo "Barnet heter ".$familie['barn']."< br >";

Sortere arrays: tilleggsstoff

Det finnes flere innebygde metoder for å sortere arrays:

Metode Forklaring
sort() sorterer indekserte arrays i stigende rekkefølge
rsort() sorterer indekserte arrays i synkende rekkefølge
asort() sorterer assosiatoive arrays i stigende rekkefølge etter verdi
ksort() sorterer assosiatoive arrays i stigende rekkefølge etter nøkkel
arsort() sorterer assosiatoive arrays i synkende rekkefølge etter verdi
krsort() sorterer assosiatoive arrays i synkende rekkefølge etter nøkkel

Eksempel: Sortere familie-arrayen i alfabetisk rekkefølge etter navn
   asort($familie);
   var_dump($familie);

Oppgaver

1. Lag en assosiativ array som inneholder fem ulike dyrearter som nøkler og navnene deres som verdi. Skriv ut en og en verdi ved å referere til nøkkelen.
2. Opprett en indeksert array som inneholder tallverdier. Sorter arrayen i stigende og i synkende rekkefølge.
3. Opprett en assosiativ array som inneholder tekstverdier. Sorter arrayen i stigende og i synkende rekkefølge etter verdier og etter nøkler
