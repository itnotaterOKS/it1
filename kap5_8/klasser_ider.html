Hva bruker vi klasser/ider til?

Det er ikke alltd vi ønsker å endre alle elementer av samme type i css:
Ikke alle < p >, ikke alle < img >, ikke alle < h2 > osv.
For å endre kun én av dem må vi gjøre den unik. Det gjør vi ved å tilordne den en identifikasjon, enten en klasse eller id.
Tilordningen skjer i taggen: f.eks. ved < h1 id="overskrift 1"> Overskrift < /h1 >. Stilsettingen for denne i css gjøres da ved å kalle en id med #:

   #overskrift1{
     color: red;
   }

Tilsvarende brukes klasser, ved < h1 class="overskrifter" > Overskrift < /h1 >. Stilsettingen for klasser i css gjøres ved å kalle klassen med .:

   .overskrifter{
     font-size: 150%;
   }

Klasse eller id?

Hovedreglen er at vi bruker klasser dersom det tilhører flere elementer, f.eks. alle overskrifter, alle bilder osv. Id bruker vi dersom det bare er én: dersom en enkelt tabellcelle skal være gul, ett enkelt avsnitt i kursiv osv.
Dersom du er usikker: bruk kun en av dem.

Kan et element ha både klasse og id?

JA! Et element kan bare ha én id, men det kan ha flere klasser eller en kombinasjon av én id og en eller flere klasser. Og det kan være en hensiktsmessig bruk av klasser og ider: Vi bruker klasser for å sette alle overskrifter til større størrelse og ider for å gi enkeltoverskrifter farge:

I html:
   < h1 > Overskrift uten noen stilsetting < /h1 >
   < h1 class="overskrift" id="over1" > Overskrift 1 < /h1 >
   < h2 class="overskrift" id="over2" > Overskrift 1 < /h2 >
   < h2 class="overskrift" id="over3" > Overskrift 1 < /h2 >
I css:
   .overskrift{
     font-size: 300%;
   }
   #over1{
     color: red;
   }
   #over2{
     color: green;
   }
   #over3{
     color: blue;
   }
