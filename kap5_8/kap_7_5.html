Ulike måter å lage menyer på

Det finnes flere ting man kan ta utgangspunkt i for å lage en god meny for et nettsted: lister, tabeller eller div-elementer. Læreboka gjennomgår listemenyer, vi kommer til å lage menyer med utgangspunkt i både lister, tabeller og div-elemeter.
Menyen kan enten være vannrett eller loddrett orientert. Vi tar for oss vannrette menyer først:

Vannrett meny med liste

Denne var vi gjennom forrige time:

  • Listepunkt
  • Listepunkt
  • Listepunkt

I html:
   < ul id="menyliste" >
     < li > < a href="#" > Listepunkt < /a > < /li >
     < li > < a href="#" > Listepunkt < /a > < /li >
     < li > < a href="#" > Listepunkt < /a > < /li >
   < /ul >

I css:
   #menyliste li {
     list-style-type: none;
     background-color: #00868B;
     color: #E5FAFB;
     width: 120px;
     margin: 3px;
     padding: 2px;
     text-align: center;
     border-radius: 10px; /* for å runde av hjørnene */
     display: inline;
   }

   #menyliste li a, #menyliste li a:visited{
     color: #E5FAFB;
     text-decoration: none;
   }

   #menyliste li a:hover{
     color: #A1C8C8;
   }

Vannrett meny med tabell

Når vi lager en tabell med lenker, vil denne kun ha en rad. Vi kan velge å ha tomme celler mellom lenkene for at det skal se mer oversiktlig ut.

Lenke Lenke Lenke Lenke

I html:
   < table id="lenketabell" >
     < tr >
     < td class="tom" > < /td >
     < td class="lenkecelle" > < a href="#" > Lenke < /a > < /td >
     < td class="tom" > < /td >
     < td class="lenkecelle" > < a href="#" > Lenke < /a > < /td >
     < td class="tom" > < /td >
     < td class="lenkecelle" > < a href="#" > Lenke < /a > < /td >
     < td class="tom" > < /td >
     < td class="lenkecelle" > < a href="#" > Lenke < /a > < /td >
     < td class="tom" > < /td >
     < /tr >
   < /table >

I css:
   #lenketabell{
     width: 90%;
     margin-left: 5%;
     height: 80px;
     border: none;
   }
   .tom{
     border: none;
     width: 4%;
   }
   .lenkecelle{
     width: 20%;
     text-align: center;
     background-color: #00868B;
     color: #E5FAFB;
     border: none;
     border-radius: 20%;
   }
   .lenkecelle a, .lenkecelle a:visited{
     color: #E5FAFB;
     text-decoration: none;
   }
  .lenkecelle a:hover{
     color: #A1C8C8;
   }

Vannrett meny med div

Menyer med div-elementer likner mye på tabellmenyer, bortsett fra at vi lar "< tr >" erstattes med en ytre < div > og "< td >" erstattes med indre div-elementer:

Lenke
Lenke

I html:
   < div id="menyrad" >
    < div class="tomDiv" > < /div >
   < div class="menyDiv"> < a href="#" > Lenke < /a > < /div >
    < div class="tomDiv" > < /div >
   < div class="menyDiv"> < a href="#" > Lenke < /a > < /div >
    < div class="tomDiv" > < /div >
   < div class="menyDiv"> < a href="#" > Lenke < /a > < /div >
    < div class="tomDiv" > < /div >
   < /div >

I css:
   #menyrad{
     height: 80px;
     width: 90%;
     margin-left: 5%;
   }
   .tomDiv{
     width: 6%;
     display: inline-block; /* må bruke inline-block for å kunne endre høyde og bredde*/
   }
   .menyDiv{
     height: 78px;
     width: 24%;
     display: inline-block;
     line-height: 78px; /* for å sentrere teksten */
     text-align: center;
     background-color: #00868B;
     color: #E5FAFB;
     font-size: 20px;
     border-radius: 15px;
   }
   .menyDiv a, .menyDiv a:visited{
     background-color: #00868B;
     color: #E5FAFB;
     text-decoration: none;
   }
   .menyDiv a:hover{
     color: #A1C8C8;
   }

Loddrett meny med liste

Lages på samme måte som vannrett meny med liste, men her utelater vi
   display: inline;

Loddrett meny med tabell

Lages på samme måte som vannrett meny med tabell, men vi får kun et < td > i hver rad, og mange < tr >

Loddrett meny med div

Lages på samme måte som vannrett meny med div, men her trenger vi ikke skrive
   display: inline-block;

Generelt om loddrette menyer

Dersom vi ønsker innhold ved siden av div'en, må vi passe på at den ytre div'en har display: inline-block:

Legger inn div-elementer for hver lenke

Legger inn alt innhold på siden

I html:
   < div id ="ramme" >
     < div id="loddrettMeny" >
       Legger inn menyen
     < /div >
     < div id="siden" >
      Legger inn alt innhold på siden
     < /div >
   < /div >

I css:
   #ramme{
     width: 100%;
     height: 300px;
     border: 1px solid black;
  }
   #loddrettMeny{
     display: inline-block;
     width: 29%;
     height: 300px;
   }
   #siden{
   display: inline-block;
   width: 70%;
   height: 300px;
   }

Utfordringer med tabell og div

På de fleste områder kommer vi lenger med å lage menyer med tabeller eller div. Ulempen med dette er at kun lenketeksten fungerer som lenke, ikke hele boksen. Dette kan vi med div-elementer ordne med å bytte rekkefølge på < div > og < a >:

Lenke
Lenke

   < div class="menyDiv" > < a href="#" > Lenke < /a > < /div >
   < a href="#" > < div class="menyDiv" > Lenke < /div> < /a >

Konklusjon: det er ofte best å lage menyer med div-elementer, her kan vi kontrollere det vi vil.

Oppgaver

1. Lag en vannrett meny med 4 lenker. Du "velger selv" metode. Menyen skal stilsettes i css slik at f.eks. skriftfarge, bakgrunnsfarge, padding og musepeker endres ved hover og active
2. Lag en loddrett meny med 6 lenker. Du "velger selv" metode. Menyen skal stilsettes i css slik at f.eks. skriftfarge, bakgrunnsfarge, padding og musepeker endres ved hover og active

Ekstra: Start en ny html-fil som du kaller mal.html eller liknende. I denne fila skal du lenke til css-fila mal.css. Legg til i html:
   Et div-element som ligger rundt alt innholdet på siden. Gi det iden "ramme" eller liknende.
   Et div-element med tekst i som fungerer som en header.
   En meny, enten med vannrett utseende eller loddrett utseende
     (lag gjerne begge muligheter og kommenter ut en av dem)
   Et div-element med iden "innhold" eller liknende.
   I denne kan du prøve deg frem med å legge inn ulikt html-innhold.
I css-fila legger du til stilsetting for de ulike div-elementene og kommenterer hva hver enkelt egenskap har å si for resultatet (se på oppgaven som en mal som inneholder alt du vil få bruk for til nettsider videre i faget)
